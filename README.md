A backend application using ruby '2.6.2' and 'rails', '~> 6.1.3' with fields and an admin panel which provides data using an API.

#Run it 
  git clone 
    cd PresApi/
      bundle update && bundle 
        rails db:migrate
          rails db:seed
            rails s
